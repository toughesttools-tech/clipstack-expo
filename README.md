## Description

ClipStack is an open-source, easy-to-use clipboard manager for iOS devices built with React Native and Expo. It allows you to store, search, and manage your clipboard history right from your mobile device.

## Features 🎯

- Store multiple clipboard entries
- Quick access to recent clipboard items
- Search and filter clipboard history
- Securely save sensitive data
- Compatible with iOS devices using Expo Go or Xcode for native builds

## Tech Stack 🚀

- React Native
- Expo (for easier setup and deployment)
- Redux for state management
- React Navigation for navigation between screens
- TypeScript for type checking and static typing

## Setup Instructions 📝

To get started with ClipStack, follow the steps below:

1. Install Node.js (https://nodejs.org/)
2. Install Expo CLI globally: `npm install -g expo-cli`
3. Create a new project using Expo: `expo init clipstack`
4. Navigate to the project folder: `cd clipstack`
5. Link ClipStack dependencies: `npm install`
6. Start the development server: `expo start`
7. (Optional) For native builds, follow the steps outlined in the Expo documentation: https://docs.expo.dev/buzz/building-standalone-apps/

## Environment Variables 🌱

ClipStack doesn't require any environment variables at this time.
